Easy way to redirect post to another post

== Description ==

 Easy way to redirect post to another post in same web/blog domain. 
= 3 Features: =
 1. Redirect to another post in same category or not 
 2. Redirect to previous/next post 
 3. Enable free to set time for redirect 

= Developed by = [ridhofitra](http://ridhofitra.info "http://ridhofitra.info")

== Installation ==

1. Upload the `Redirect Post` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Setting -> Redirect Post, Click on Redirect Post
